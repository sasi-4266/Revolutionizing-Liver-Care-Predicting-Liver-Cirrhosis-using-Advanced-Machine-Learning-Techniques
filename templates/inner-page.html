Revolutionizing Liver Care

Project: Liver Cirrhosis Prediction Using Machine Learning

This project aims to build a machine learning model to predict liver cirrhosis using clinical features such as bilirubin levels, enzyme activity, albumin, and diabetes indicators.

Objectives

  • Early detection of liver cirrhosis through data-driven insights
  • Use machine learning models like Random Forest for prediction
  • Deploy a user-friendly web app using Flask

Tools and Technologies

  • Python (Pandas, Scikit-learn)
  • Flask for web deployment
  • HTML, CSS for UI
  • Excel dataset: HealthCareData.xlsx

Outcome

A working liver cirrhosis prediction system integrated with a web interface. Users can input patient data and get instant predictions with a model trained on real clinical indicators.
