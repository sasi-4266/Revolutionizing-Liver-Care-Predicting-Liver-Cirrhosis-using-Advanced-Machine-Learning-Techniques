Liver Cirrhosis Prediction

Project Portfolio Details

This machine learning project predicts liver cirrhosis based on clinical inputs such as enzyme levels and diabetic factors. A Random Forest model was trained on real patient data with a focus on early detection and real-world usability.

Key Features

  • Medical dataset: HealthCareData.xlsx
  • Exploratory Data Analysis (EDA) and visualization
  • Random Forest model (68% accuracy)
  • Web interface using Flask
  • Prediction form for real-time results
